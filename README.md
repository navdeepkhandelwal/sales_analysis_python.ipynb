# 📊 Sales Data Analysis (Superstore Dataset)

## 📌 Project Overview
This project focuses on analyzing retail sales data using Python. The goal is to perform data cleaning, exploratory data analysis (EDA), and visualization to uncover business insights related to sales, profit, and customer trends.

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
- Superstore Sales Dataset (Excel format)
- Contains information on orders, customers, sales, profit, and regions

---

## 🔍 Key Steps Performed

### 1. Data Cleaning
- Removed missing values and duplicates  
- Converted date columns to proper datetime format  

### 2. Feature Engineering
- Extracted Month and Year from order date  

### 3. Exploratory Data Analysis (EDA)
- Analyzed total sales and profit  
- Category-wise and region-wise performance  
- Identified top-selling products  

### 4. Data Visualization
- Sales by Category (Bar Chart)  
- Monthly Sales Trend (Line Chart)  
- Region-wise Sales Distribution (Pie Chart)  
- Profit vs Sales (Scatter Plot)  

---

## 📈 Key Insights
- Technology category generates the highest sales  
- Certain regions contribute more to overall revenue  
- High sales do not always result in high profit  
- Discounts and pricing impact profitability  

---
## 🎯 Conclusion
This project demonstrates how Python can be used to analyze business data and generate actionable insights through data cleaning, analysis, and visualization.
